# BioPredict: Advanced Body Performance Analytics & ML Modeling 🏋️‍♂️📊

## 📌 Project Overview
This is a comprehensive, end-to-end Data Science project designed to analyze physical performance metrics and build high-accuracy predictive models. The project stands out by implementing a **dual-modeling approach**: 
1. **Multi-class Classification:** To categorize individuals into four fitness grades (A, B, C, D).
2. **Regression Analysis:** To predict specific physical outcomes, such as broad jump distance.

## 🚀 Key Technical Highlights
- **Dual Machine Learning Approach:** - **Classification:** Compared **7 different models** (including Random Forest, Gradient Boosting, and SVM) to find the best predictor for performance classes.
    - **Regression:** Evaluated **8 regression models** to precisely predict numerical physical metrics.
- **Deep Exploratory Data Analysis (EDA):** Leveraged Seaborn and Matplotlib to visualize complex correlations between body fat, age, and physical strength.
- **Robust Preprocessing:** Implemented advanced data cleaning, outlier handling, and feature encoding to ensure model stability and reliability.
- **Performance Evaluation:** Utilized **5+10-fold Cross-Validation**, Confusion Matrices, and R² scoring for rigorous model validation.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn.
- **Environment:** Google Colab / Jupyter Notebook.

## 📊 Dataset Insights
The analysis was performed on the **Body Performance Dataset** (13,393 records), identifying key predictors of physical excellence:
- **Top Predictors:** Broad Jump, Sit-ups counts, and Grip Force.
- **Key Finding:** Body Fat percentage was identified as a primary barrier to reaching "Class A" performance levels.

## 📂 Project Structure
- `Data Exploration:` Statistical analysis and visualization of 12 physical features.
- `Feature Engineering:` Data normalization and categorical encoding.
- `Model Comparison:` Benchmarking multiple algorithms to select the optimal model.
- `Results & Evaluation:` Detailed reports on accuracy, precision, and error metrics.

## 🏁 Conclusion
This project demonstrates the ability to handle complex datasets and apply both classification and regression techniques to solve real-world analytical problems in the health and fitness domain.
